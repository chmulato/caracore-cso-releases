Acesse GitHub Releases

O link aponta dinamicamente para a release mais recente. Certifique-se de baixar a versão mais atual.

Requisitos do Sistema

  • Sistema Operacional: Windows 10/11 (64 bits)
  • Processador: Intel Core i3 ou equivalente (2 GHz)
  • Memória RAM: 4 GB mínimo (8 GB recomendado)
  • Espaço em Disco: 500 MB para instalação + espaço para dados
  • Java: JDK 21 incluído no instalador

Como Instalar

  1. Baixe o arquivo CaraCore-CSO-Setup.exe das releases
  2. Execute o instalador como administrador
  3. Siga as instruções na tela
  4. A aplicação será instalada em C:\Program Files\CaraCore CSO\
  5. Use o atalho criado na área de trabalho para iniciar

Verificação de Integridade

Para garantir que o download não foi corrompido, verifique o checksum SHA-256:

  • Baixe também o arquivo SHA256SUMS da release
  • Use o comando: certutil -hashfile CaraCore-CSO-Setup.exe SHA256
  • Compare com o valor no arquivo SHA256SUMS
