O Cara Core CSO será distribuído como instalador .exe para Windows. O instalador inclui Java 25 e todas as dependências necessárias. A aplicação usa JavaFX 25 para a interface gráfica e SQLite para armazenamento local.

Como executar (desenvolvimento):

  • Na oficina (caracore-cso): cd src && mvn javafx:run
  • Ou gerar instalador: scripts de build para Windows

Requisitos do Sistema

  • Sistema Operacional: Windows 10/11 (64-bit)
  • Processador: Intel/AMD x64
  • Memória RAM: 8 GB mínimo, 16 GB recomendado
  • Armazenamento: 500 MB para instalação + espaço para dados
  • Permissões: Instalação requer privilégios de administrador

Verificação de Integridade

Todos os releases incluem checksums SHA-256 para verificação de integridade. Após o download, verifique o arquivo usando:

Get-FileHash -Path .\CSO-Setup.exe -Algorithm SHA256

Compare o resultado com o hash publicado no release.
