Cara Core CSO

Gestão de Transportes

O Produto

O Cara Core CSO é uma aplicação JavaFX 25 desenvolvida para transportadoras e centros de distribuição. Com dados armazenados localmente em SQLite, garante funcionamento offline completo e controle total sobre as informações.

Gestão de Entregas

Controle completo do ciclo de vida das entregas: cadastro, status, tracking e finalização.

Couriers e Equipes

Cadastro e organização de entregadores, com controle de performance e disponibilidade.

Clientes e CDs

Administração de centros de distribuição e clientes com perfis completos.

Preços e Tabelas

Controle de tabelas de preços, fretes e custos operacionais.

SMS/WhatsApp

Integração para notificações automáticas de status de entregas.

Relatórios

Dashboards e relatórios para análise de performance operacional.

Especificações Técnicas

Plataforma: Windows Desktop
Tecnologia: Java 25 + JavaFX
Banco de Dados: SQLite
Modo: Offline-first
Instalador: .exe (Windows)
Verificação: SHA-256 checksums
