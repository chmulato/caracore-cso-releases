# Manual do CaraCore CSO

## Objetivo do Manual

Este manual apresenta o uso funcional do CaraCore CSO com foco em operação, instalação e proteção de dados.

## Escopo

O documento cobre:
1. Visão geral da solução.
2. Instalação e configuração inicial.
3. Diretrizes de proteção de dados.
4. Práticas de resiliência offline.

## Visão Geral do Produto

O CaraCore CSO é uma aplicação desktop para gestão de transportadoras, com suporte a controle de entregas, gestão de couriers, administração de clientes e controle de preços.

Base técnica:
1. Java 25.
2. JavaFX 25.
3. SQLite local.
