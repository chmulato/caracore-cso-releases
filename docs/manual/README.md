# Manual do CaraCore CSO

Bem-vindo ao manual da aplicação CaraCore CSO. Aqui você encontrará informações completas sobre como usar o sistema de gestão para transportadoras e entender suas funcionalidades.

## O que é o CaraCore CSO?

O CaraCore CSO é um sistema desktop profissional para gestão completa de transportadoras, oferecendo controle total sobre entregas, couriers, clientes e preços. Desenvolvido em Java 25 + JavaFX, com dados armazenados localmente em SQLite para máxima segurança e performance.
