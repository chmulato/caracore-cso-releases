# Instalação e Configuração

## Requisitos Mínimos

1. Sistema operacional Windows 10 ou Windows 11 (64 bits).
2. Memória RAM de 4 GB (8 GB recomendados).
3. Espaço em disco de 500 MB para instalação, além da base de dados operacional.

## Fonte Oficial de Download

1. Loja: https://cso.caracore.com.br/.
2. Releases: https://github.com/chmulato/caracore-cso-releases/releases.

## Procedimento de Instalação

1. Acesse o canal oficial e baixe o instalador da versão desejada.
2. Execute o instalador com privilégios de administrador.
3. Siga o assistente de instalação até a conclusão.
4. Abra o sistema pelo atalho criado no menu do Windows.

## Verificação Pós-Instalação

1. Confirmar abertura normal da aplicação.
2. Confirmar criação da base local SQLite.
3. Validar acesso ao módulo inicial sem erros.

## Configuração Inicial Recomendada

1. Definir preferências locais de operação.
2. Revisar parâmetros de backup local.
3. Validar rotinas de uso offline antes da entrada em produção.
