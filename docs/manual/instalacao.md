# Instalação

## Requisitos

- Java 25
- Sistema Operacional: Windows, Linux ou macOS

## Passos de Instalação

1. Baixe a release mais recente do GitHub Releases.
2. Execute o instalador.
3. Siga as instruções na tela.

## Configuração Inicial

Após a instalação, configure suas preferências locais para otimizar a resiliência offline.
