# Proteção de Dados

## Princípios

A proteção de dados do CaraCore CSO segue os princípios de confidencialidade, integridade e disponibilidade operacional local.

## Controles Aplicados

1. Criptografia de dados em repouso e em trânsito, quando aplicável ao fluxo.
2. Controle de acesso local por perfil de usuário.
3. Rotinas de backup local para recuperação operacional.

## Diretrizes de Uso

1. Manter o ambiente do sistema com acesso restrito a operadores autorizados.
2. Executar backups periódicos conforme política interna da operação.
3. Validar regularmente a restauração de backup em ambiente de teste.

## Conformidade

A aplicação prioriza soberania de dados por meio de armazenamento local em SQLite, reduzindo dependência de serviços externos para operações críticas.
