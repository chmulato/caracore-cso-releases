# Proteção de Dados

## Estratégias de Proteção

O CSOnline utiliza várias camadas de proteção para garantir a segurança dos dados:

- **Criptografia**: Dados criptografados em repouso e em trânsito.
- **Controle de Acesso**: Apenas o usuário local tem acesso aos dados.
- **Backups Locais**: Possibilidade de backups manuais.

## Conformidade

A aplicação segue princípios de soberania de dados, evitando dependências de nuvens externas.
